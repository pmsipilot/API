API PMSIpilot

L'expertise de PMSIpilot à la disposition de tous.

Informations du SIH en temps-réel.
Accédez à tous les patients de votre établissement, aux séjours en cours ou passés ainsi qu'à l'ensemble des actes réalisés.

Référentiels CCAM et CIM10.
Utilisez à volonté les référentiels CCAM et CIM10 de PMSIpilot : ils sont historisés et régulièrement mis à jour selon la réglementation.

Les API PMSIpilot sont disponibles pour tous les clients PMSIpilot.
Elles peuvent être utilisées par n'importe quel logiciel tiers, quel qu'en soit l'éditeur.
L'établissement peut aussi utiliser les API PMSIpilot pour réaliser des applications "maisons".
