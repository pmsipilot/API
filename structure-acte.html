Structure d'un acte

Liste des champs disponibles :

  • iep obligatoire identifiant unique du séjour auquel appartient le mouvement.
  • ipa obligatoire identifiant permanent et unique de l'acte.
  • type norme CCAM, CDARR ou NGAP
  • code
  • date_realisation norme aaaa-mm-jj hh:mm:ss
  • activite
  • phase
  • code_intervenant
  • numero_adeli

Exemple en JSON :

{
  "iep": "0000311963",
  "ipa": "16871",
  "type": "CCAM",
  "code": "LJMA002",
  "date_realisation": "2012-01-11 12:21:13",
  "activite": "1",
  "phase": "0",
  "code_intervenant": "22",
  "numero_adeli": "984357124567897"
}
