Structure d'une identité

Liste des champs disponibles :

  • ipp obligatoire identifiant unique de l'identité.
  • insc
  • nom_usage
  • nom_naissance
  • pseudonyme
  • prenom
  • sexe norme H homme - F femme - I indéterminé - A autre
  • date_naissance norme aaaa-mm-jj hh:mm:ss
  • adresse_1
  • adresse_2
  • commune
  • code_postal
  • cedex
  • pays
  • telephone_domicile
  • telephone_portable
  • telephone_pro
  • date_creation norme aaaa-mm-jj hh:mm:ss

Exemple en JSON :

{
  "ipp": "1106000987",
  "insc": null,
  "nom_usage": null,
  "nom_naissance": "TEST",
  "pseudonyme": null,
  "prenom": "CLIMCO",
  "sexe": "H",
  "date_naissance": "1970-01-01 00:00:00",
  "adresse_1": "BP 525",
  "adresse_2": null,
  "commune": "AMBILLY",
  "code_postal": "74100",
  "cedex": null,
  "pays": "100",
  "telephone_domicile": "+33(0)499999999",
  "telephone_portable": null,
  "telephone_pro": null,
  "date_creation": "2011-12-13 15:37:10"
}
