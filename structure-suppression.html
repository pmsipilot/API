Structure d'une suppression

Liste des champs disponibles :

  • resource_name type de la resource supprimée.
  • resource tableau au format JSON représentant la resource supprimée.

Exemple en JSON :

{
  "resource_name": "actes",
  "resource":
  {
    "iep": "0000311963",
    "ipa": "16871",
    "type": "CCAM",
    "code": "LJMA002",
    "date_realisation": "2012-01-11 12:21:13",
    "activite": "1",
    "phase": "0",
    "code_intervenant": "22",
    "numero_adeli": "984357124567897"
  }
}
