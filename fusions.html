API fusions

Récupération de la liste des fusions

GET api/fusions?token=#{token}

Si le paramètre token est spécifié, seules les fusions ajoutées ou modifiées depuis ce token seront retournées.

L'API retourne la liste des fusions triées par date de création (les plus anciennes en premier).

Récupération d'une fusions

GET api/fusions/#{ipf}

Le paramètre obligatoire ipf permet de cibler la fusion à retourner.

L'API retourne un code 200 ("OK") et la fusion demandée.

Ajout d'une fusion

POST api/fusions

Une fusion doit être passée en paramètre.

L'API retourne un code 201 ("Created").
